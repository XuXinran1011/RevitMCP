## 基本概念

RevitMCP基于Model Context Protocol (MCP)，它允许大型语言模型(LLM)与Revit进行安全、受控的交互。以下是一些关键概念：

### MCP工具

RevitMCP提供了一系列工具，使LLM能够执行特定的Revit操作：

- **查询工具**：用于从Revit模型中检索信息
- **修改工具**：用于修改Revit模型中的元素和参数
- **创建工具**：用于在Revit模型中创建新元素
- **视图工具**：用于管理和操作Revit视图

### 上下文理解

RevitMCP能够理解BIM上下文，这意味着它可以：

- 理解建筑和工程术语
- 识别Revit特定的概念和对象
- 将自然语言请求转换为精确的Revit操作

## 入门

### 基本交互

一旦您[安装并配置](INSTALLATION.md)了RevitMCP，您就可以开始使用自然语言与Revit交互。以下是一些基本示例：

#### 查询元素

```
请列出当前模型中的所有墙
```

这将返回当前Revit模型中所有墙的列表，包括它们的ID、类型和基本属性。

#### 获取元素属性

```
告诉我ID为123456的墙的属性
```

这将返回指定墙的详细属性，如材料、尺寸、防火等级等。

#### 修改参数

```
将ID为123456的墙的高度更改为4米
```

这将修改指定墙的高度参数。

### 复杂查询

RevitMCP支持更复杂的查询，允许您筛选和分析Revit数据：

```
找出所有高度大于3米且使用"基本墙"类型的墙
```

```
计算当前模型中所有门的总数，并按类型分组
```

## 常见用例

### 设计查询和分析

RevitMCP可以帮助设计师快速获取关于其模型的信息：

```
分析当前楼层的空间使用效率，计算可用面积与总面积的比率
```

```
检查所有外墙的隔热性能，并标识可能需要改进的区域
```

### 模型修改

您可以使用RevitMCP进行各种模型修改：

```
在坐标(10,15,0)处创建一个2.1米x0.9米的单扇门
```

```
将所有内墙的防火等级更新为"2小时"
```

### 文档和视图管理

RevitMCP可以帮助管理Revit视图和文档：

```
创建一个名为"结构分析"的新平面视图，显示所有结构元素
```

```
生成当前模型的所有外墙的材料清单
```

## 高级功能

### 批量操作

RevitMCP支持批量操作，允许您一次修改多个元素：

```
将所有会议室的天花板高度增加30厘米
```

### 条件逻辑

您可以使用条件逻辑来执行更复杂的操作：

```
找出所有没有防火门的防火分区，并在每个分区的出口处添加防火门
```

### 工作流自动化

RevitMCP可以自动化复杂的工作流程：

```
为每个房间创建家具布置方案，确保符合建筑规范并优化空间使用
```

## 最佳实践

### 明确指令

为获得最佳结果，请尽量明确您的指令：

- **指定元素**：通过ID、名称或明确的属性指定元素
- **提供上下文**：包括相关的背景信息
- **定义参数**：明确指定单位和值

### 迭代方法

复杂任务通常最好分解为多个步骤：

1. 首先查询以获取必要的信息
2. 分析结果
3. 执行具体的修改操作
4. 验证更改

### 安全考虑

使用RevitMCP时请记住以下安全注意事项：

- **备份模型**：在进行重大更改前备份您的Revit模型
- **验证更改**：检查RevitMCP所做的更改是否符合预期
- **权限控制**：了解RevitMCP的权限限制

## 故障排除

### 常见问题

#### 命令未被理解

**问题**：RevitMCP没有按预期执行命令
**解决方案**：
- 尝试重新表述您的请求，使用更明确的术语
- 确保您的请求在RevitMCP的能力范围内
- 检查是否有任何语法或逻辑错误

#### 操作失败

**问题**：RevitMCP尝试执行操作但失败
**解决方案**：
- 检查Revit模型是否处于可编辑状态
- 验证您引用的元素是否存在
- 确保操作符合Revit的规则和约束
